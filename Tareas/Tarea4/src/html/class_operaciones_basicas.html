Detailed Description

template<typename T>
class OperacionesBasicas< T >

La clase OperacionesBasicas proporciona metodos para validar y realizar operaciones matriciales basicas, incluyendo suma, resta y multiplicacion. Estos metodos toman dos matrices como parametros y verifican si cumplen con las condiciones necesarias para realizar la operacion correspondiente antes de realizarla. Cada operacion calcula el resultado y lo imprime en la consola.
